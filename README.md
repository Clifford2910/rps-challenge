# RPS Game

## Setup instructions

To install and run:

```
$ git clone https://github.com/Clifford2910/rps_challenge.rb.git
$ cd rps_challenge.rb
$ bundle
$ rackup
```

Then in your browser visit:   
```
localhost:9292
```



## User stories


## Game rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
